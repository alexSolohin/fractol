# fractol

## Project Overview

- The project is about creating beautiful fractals.

## How to compile:
- git clone https://github.com/alexSolohin/fractol.git
- cd fractol
- make

## How to run
- ./fractol { fractals }

## Fractals
- mandelbrot
- julia
- burningship
- mandelbar
- celtic_mandelbrot
- celtic_mandelbar
- celtic_perpendicular
